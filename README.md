# Blog Zone Api
    - Simple blog website powered by NodeJS - ReactJs - MongoDb 

# Installation
    - Clone the project

    - Create .env file

    - Add 
        MONGO_URI='your_db_uri'
        JWT_SECRET='your_jwt_secret'
        CLIENT_URL='blog_zone_client_url'
        NODE_ENV='dev mode'

    - Run
```bash
npm start
```

# Usage 
    - Blog_zone client is running on port 3000
